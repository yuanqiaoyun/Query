在组件中拿到store中的值,看是什么情况了，
如果是拿到store中的state的话，可以是在计算属性中去拿。
computed：{
  count(){
    return this.$store.state.count;
  }
},

同样的mutations中的东西的调用也是有两种方法，1中是import 然后利用ES6的语法来直接展开，
然后就相当于是这个组件的东西了。
例如 拿mutstions中的方法，

在组件中 先引入
import {mapMutations} from vuex;
然后如果是属性的话就在competed中展开，如果是方法的话就在methods的话就在methonds中对应展开。
methods:{
  ...mapMutations(['方法名1']，['方法名2']),
}
